什么时候会产生这个错误？

JSLint在遇到function关键字而它会被解析成一个紧随着一个打开的圆括号的语句时，会抛出"Missing name in function declaration"的错误。在下面例子中，我们尝试着定义一个函数，但是并没有给予它标示符。

这个与JSLint中的 "Missing name in function statement"是等价的。更多的细节可以从该JSLint消息的页面中找到。

在JSHint1.0.0及以上版本中，你可以通过一个 特殊的选项语法来忽略该警告。这个警告的标示符是W025。这也就意味着你可以使用/*jshint -W025 */来告诉JSHint不要报告这个警告。
