为什么会产生这个错误？

JSHint (版本1.0.0及以上) 在遇到一个函数的形参数量比maxparams选项声明的要多时，会抛出"Too many parameters per function ({a})"的错误。以下是一个我们尝试着声明一个带有三个参数的函数的例子：

这个与老版本的JSHint"Too many parameters per function"警告是相同的。更多细节可以在该页面上找到。

在JSHint1.0.0及以上版本中你可以通过特殊的语法来忽略任意警告。这个警告的标示符是W072。这意味着你可以通过/*jshint -W072*/让其不要报告该警告。
